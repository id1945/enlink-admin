Introduction

Enlink is a modern dashboard template build with Angular & Ng-Zorro. It contains alot high flexibility and configurablue components, which written in TypeScript with complete defined types. With a detailed documentation & clean code, it allow developers build any projects easily!

Installation

Enlink is created based on Angular, it required some prerequisite before starting the app.

Node & NPM

You can download Node.js from https://nodejs.org/en/download/. NPM comes bundled with Node.js.

Angular CLI

Run the command below at your cmd as Administrator to install Angular Cli, make sure npm was installed before installing Angular CLI

npm install -g @angular/cli
NPM Package

Run npm install from the root of your project(where package.json located) to install all the necessary dependencies.

Development Server

Run npm run start or ng serve -o to start a local dev server

Others Dev Command
COMMAND DESCRIPTION
npm run start-aot Run npm run start in aot mode.
Production

Run ng build --prod to build the app for production, the build artifacts will be stored in the dist/ directory

Others Build Command
COMMAND DESCRIPTION
npm run build Run ng build --prod with base href ./ & max_old_space_size=8000
npm run build-prod-state Run npm run build and create a state.json for bundle analayze.
npm run bundle-report Create a bundle report to analayze app size.

Routing

Enlink comes with 2 typess of layout, which is common-layout & full-layout, layouts contain the main view of your app, these can be found in directory src\app\layouts

  • - common-layout: contain pre built template components.
  • - full-layout: contain only clean view.

Both layouts have their own routing file, which located at src\app\shared\routes directories, layouts load sub modules instead of single component.
